  <div className="text-center mb-12">
    <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get oxidized-cell running in minutes. Follow these steps to build and launch the emulator.
    </p>
  </div>

  <Steps>
    <Step title="Install dependencies">
      Install Rust 1.70+, a C++17 compiler, CMake, LLVM 14+, and Vulkan SDK 1.2+.
      
      ```bash Ubuntu/Debian
      sudo apt update
      sudo apt install -y build-essential cmake llvm-dev libvulkan-dev libasound2-dev
      ```
    </Step>

    <Step title="Clone and build">
      Clone the repository and build with Cargo in release mode for optimal performance.
      
      ```bash
      git clone https://github.com/darkace1998/oxidized-cell.git
      cd oxidized-cell
      cargo build --release
      ```
    </Step>

    <Step title="Install PS3 firmware">
      Download and install the official PS3 firmware to decrypt commercial games.
      
      ```bash
      ./scripts/download-firmware.sh
      ```
      
      <Note>
        The PS3 System Software is free and legal to download from Sony's servers. This is required for running encrypted commercial games.
      </Note>
    </Step>

    <Step title="Run the emulator">
      Launch the emulator with the UI or load a game directly.
      
      ```bash
      # Launch UI mode
      cargo run --release

      # Load a specific game
      cargo run --release -- /path/to/game.elf
      ```
    </Step>
  </Steps>
</div>
