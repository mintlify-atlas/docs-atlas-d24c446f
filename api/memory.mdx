title: "oc_memory"
description: "Memory management for the PS3 address space"
---

The `oc_memory` crate provides the virtual memory system that mimics the PS3's 32-bit address space, including reservation system for SPU atomics.

## Overview

Key features:

- 32-bit virtual address space management
- Page-level memory protection
- Reservation system for load-link/store-conditional (atomics)
- RSX memory mapping
- Shared memory regions between PPU and SPU
- Memory watchpoints for debugging
- Cache simulation and profiling

## MemoryManager

## Page flags

<ResponseField name="PageFlags" type="bitflags">
  Memory page protection flags
  
  ```rust
  bitflags! {
      pub struct PageFlags: u8 {
          const READ    = 0b0001;  // Readable
          const WRITE   = 0b0010;  // Writable
          const EXECUTE = 0b0100;  // Executable
          const MMIO    = 0b1000;  // Memory-mapped I/O
      }
  }
  
  // Common combinations
  const RW  = READ | WRITE;
  const RX  = READ | EXECUTE;
  const RWX = READ | WRITE | EXECUTE;
  ```
</ResponseField>
