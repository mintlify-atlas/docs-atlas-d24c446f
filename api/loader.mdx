The `oc_loader` crate handles loading PS3 executables and libraries:

- **ELF files** - Plain executables
- **SELF files** - Encrypted executables (signed/encrypted ELF)
- **PRX files** - Loadable modules (shared libraries)
- **PUP files** - Firmware updates
- **PKG files** - Game packages

## ElfLoader

Loads standard ELF64 files (PowerPC 64-bit big-endian).

### Creation

<ParamField path="new" type="fn<R: Read + Seek>(reader: &mut R) -> Result<Self, LoaderError>">
  Creates a new ELF loader by parsing the file.
  
  ```rust
  use oc_loader::ElfLoader;
  use std::fs::File;
  
  let mut file = File::open("EBOOT.BIN")?;
  let elf = ElfLoader::new(&mut file)?;
  
  println!("Entry point: 0x{:x}", elf.entry_point);
  println!("Program headers: {}", elf.phdrs.len());
  ```
</ParamField>

### Loading

<ParamField path="load_segments" type="fn<R: Read + Seek>(&self, reader: &mut R, memory: &Arc<MemoryManager>, base_addr: u32) -> Result<(), LoaderError>">
  Loads all LOAD segments into memory at the specified base address.
  
  ```rust
  use oc_memory::MemoryManager;
  
  let memory = MemoryManager::new()?;
  elf.load_segments(&mut file, &memory, 0x01000000)?;
  println!("ELF loaded at 0x01000000");
  ```
</ParamField>

## SelfLoader

Loads SELF (Signed ELF) files with decryption support.

### Creation

<ParamField path="new" type="fn() -> Self">
  Creates a new SELF loader without decryption keys.
  
  ```rust
  use oc_loader::SelfLoader;
  
  let loader = SelfLoader::new();
  ```
</ParamField>

<ParamField path="with_firmware" type="fn(firmware_path: &str) -> Result<Self, LoaderError>">
  Creates a loader with firmware keys extracted from a PUP file.
  
  ```rust
  let loader = SelfLoader::with_firmware("/path/to/PS3UPDAT.PUP")?;
  ```
</ParamField>

<ParamField path="with_keys_file" type="fn(keys_path: &str) -> Result<Self, LoaderError>">
  Creates a loader with keys from a keys.txt file.
  
  ```rust
  let loader = SelfLoader::with_keys_file("/path/to/keys.txt")?;
  ```
</ParamField>

### Decryption

<ParamField path="decrypt" type="fn(&self, data: &[u8]) -> Result<Vec<u8>, LoaderError>">
  Decrypts a SELF file and extracts the ELF data.
  
  ```rust
  use std::fs;
  
  let self_data = fs::read("EBOOT.BIN")?;
  let elf_data = loader.decrypt(&self_data)?;
  
  // Now parse as ELF
  let mut cursor = std::io::Cursor::new(&elf_data);
  let elf = ElfLoader::new(&mut cursor)?;
  ```
</ParamField>

<ParamField path="is_self" type="fn(data: &[u8]) -> bool">
  Checks if data is a SELF file (checks for SCE magic).
  
  ```rust
  if SelfLoader::is_self(&data) {
      println!("This is a SELF file");
  } else {
      println!("This is a plain ELF file");
  }
  ```
</ParamField>

<ParamField path="has_keys" type="fn(&self) -> bool">
  Checks if decryption keys are available.
  
  ```rust
  if !loader.has_keys() {
      eprintln!("Warning: No decryption keys loaded");
  }
  ```
</ParamField>

<ParamField path="verify_decryption" type="fn(&self, data: &[u8]) -> Result<bool, LoaderError>">
  Verifies that a SELF file can be decrypted with the current keys.
  
  Returns `Ok(true)` if decryption produces valid ELF data, `Ok(false)` if wrong keys.
  
  ```rust
  match loader.verify_decryption(&self_data) {
      Ok(true) => println!("Decryption successful"),
      Ok(false) => println!("Wrong keys for this SELF"),
      Err(e) => eprintln!("Error: {}", e),
  }
  ```
</ParamField>

## PrxLoader

Loads PRX (PS3 shared library) modules with import/export resolution.

### Creation

<ParamField path="new" type="fn() -> Self">
  Creates a new PRX loader.
  
  ```rust
  use oc_loader::PrxLoader;
  
  let mut loader = PrxLoader::new();
  ```
</ParamField>

### Module loading

<ParamField path="load_module" type="fn<R: Read + Seek>(&mut self, name: String, reader: &mut R, memory: &Arc<MemoryManager>, base_addr: u32) -> Result<PrxModule, LoaderError>">
  Loads a PRX module from ELF data.
  
  ```rust
  use std::fs::File;
  
  let mut file = File::open("libsysutil.prx")?;
  let module = loader.load_module(
      "libsysutil".to_string(),
      &mut file,
      &memory,
      0x90000000,
  )?;
  
  println!("Loaded {} exports, {} imports",
           module.exports.len(),
           module.imports.len());
  ```
</ParamField>

### Symbol resolution

<ParamField path="resolve_nid_to_name" type="fn(&self, nid: u32) -> Option<&str>">
  Resolves a NID (Name ID hash) to a function name using the built-in database.
  
  ```rust
  if let Some(name) = loader.resolve_nid_to_name(0x9FB6228E) {
      println!("NID 0x9FB6228E = {}", name);  // sys_ppu_thread_create
  }
  ```
</ParamField>

<ParamField path="add_nid_mapping" type="fn(&mut self, nid: u32, name: String)">
  Adds a custom NID mapping.
  
  ```rust
  loader.add_nid_mapping(0x12345678, "my_custom_function".to_string());
  ```
</ParamField>

### Stub libraries

<ParamField path="create_stub_library" type="fn(&mut self, name: String, version: u32) -> &mut StubLibrary">
  Creates a stub library for unresolved imports.
  
  ```rust
  let stub_lib = loader.create_stub_library("cellGcmSys".to_string(), 0x10000);
  ```
</ParamField>

<ParamField path="add_stub_function" type="fn(&mut self, library_name: &str, nid: u32, func_name: String, stub_addr: u32, return_value: i64)">
  Adds a stub function that returns a default value.
  
  ```rust
  loader.add_stub_function(
      "cellGcmSys",
      0x21AC3697,
      "cellGcmInit".to_string(),
      0x90000000,
      0,  // Return CELL_OK
  );
  ```
</ParamField>

## Error types

<ResponseField name="LoaderError" type="enum">
  Loader errors
  
  - `InvalidElf(String)` - Invalid ELF file
  - `InvalidSelf(String)` - Invalid SELF file
  - `DecryptionFailed(String)` - Decryption failed
  - `MissingPrx(String)` - PRX module not found
  - `MissingModule(String)` - Required module not found
  - `InvalidPkg(String)` - Invalid PKG file
  - `InvalidPup(String)` - Invalid PUP file
  - `InvalidFirmware(String)` - Invalid firmware
  - `UnsupportedFormat(String)` - Unsupported file format
</ResponseField>

## Example: Loading a game

```rust
use oc_loader::{SelfLoader, ElfLoader};
use oc_memory::MemoryManager;
use std::fs;

// Read EBOOT.BIN
let data = fs::read("/dev_hdd0/game/NPEB00000/USRDIR/EBOOT.BIN")?;

// Check if it's a SELF file
if SelfLoader::is_self(&data) {
    // Load decryption keys
    let loader = SelfLoader::with_firmware("/path/to/PS3UPDAT.PUP")?;
    
    // Decrypt SELF
    let elf_data = loader.decrypt(&data)?;
    
    // Parse ELF
    let mut cursor = std::io::Cursor::new(&elf_data);
    let elf = ElfLoader::new(&mut cursor)?;
    
    // Load into memory
    let memory = MemoryManager::new()?;
    elf.load_segments(&mut cursor, &memory, 0x01000000)?;
    
    println!("Game loaded! Entry point: 0x{:x}", elf.entry_point);
} else {
    // Plain ELF file (homebrew)
    let mut cursor = std::io::Cursor::new(&data);
    let elf = ElfLoader::new(&mut cursor)?;
    
    let memory = MemoryManager::new()?;
    elf.load_segments(&mut cursor, &memory, 0x01000000)?;
    
    println!("Homebrew loaded! Entry point: 0x{:x}", elf.entry_point);
}
```
