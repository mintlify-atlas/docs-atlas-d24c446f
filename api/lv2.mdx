The `oc_lv2` crate implements high-level emulation of the PS3's LV2 kernel (hypervisor) system calls.

## Overview

LV2 is the PS3's kernel layer that provides:

- Process and thread management
- Synchronization primitives (mutexes, condition variables, semaphores)
- Memory management
- File system operations
- Timer and time management
- SPU management
- PRX (module) loading

## SyscallHandler

The main syscall dispatcher that manages kernel state and routes system calls.

## Synchronization primitives

The `oc_lv2` crate provides full implementations of PS3 synchronization primitives.

### Mutex

```rust
use oc_lv2::sync::mutex;

// Create mutex
let mutex_id = mutex::syscalls::sys_mutex_create(
    &object_manager,
    mutex::MutexAttributes::default(),
)?;

// Lock
mutex::syscalls::sys_mutex_lock(&object_manager, mutex_id, thread_id)?;

// Unlock
mutex::syscalls::sys_mutex_unlock(&object_manager, mutex_id, thread_id)?;

// Destroy
mutex::syscalls::sys_mutex_destroy(&object_manager, mutex_id)?;
```

### Condition variable

```rust
use oc_lv2::sync::cond;

// Create condition variable
let cond_id = cond::syscalls::sys_cond_create(
    &object_manager,
    cond::CondAttributes::default(),
)?;

// Wait
cond::syscalls::sys_cond_wait(
    &object_manager,
    cond_id,
    mutex_id,
    thread_id,
    timeout_usec,
)?;

// Signal one
cond::syscalls::sys_cond_signal(&object_manager, cond_id)?;

// Signal all
cond::syscalls::sys_cond_signal_all(&object_manager, cond_id)?;
```

### Semaphore

```rust
use oc_lv2::sync::semaphore;

// Create semaphore
let sem_id = semaphore::syscalls::sys_semaphore_create(
    &object_manager,
    initial_count,
    max_count,
)?;

// Wait (decrement)
semaphore::syscalls::sys_semaphore_wait(&object_manager, sem_id, timeout)?;

// Post (increment)
semaphore::syscalls::sys_semaphore_post(&object_manager, sem_id, count)?;
```

### Event flag

```rust
use oc_lv2::sync::event_flag;

// Create event flag
let flag_id = event_flag::syscalls::sys_event_flag_create(
    &object_manager,
    event_flag::EventFlagAttributes::default(),
    init_pattern,
)?;

// Wait for pattern
event_flag::syscalls::sys_event_flag_wait(
    &object_manager,
    flag_id,
    pattern,
    mode,  // AND or OR
    timeout,
)?;

// Set bits
event_flag::syscalls::sys_event_flag_set(&object_manager, flag_id, pattern)?;
```

### RwLock (reader-writer lock)

```rust
use oc_lv2::sync::rwlock;

// Create rwlock
let rwlock_id = rwlock::syscalls::sys_rwlock_create(
    &object_manager,
    rwlock::RwLockAttributes::default(),
)?;

// Read lock
rwlock::syscalls::sys_rwlock_rlock(&object_manager, rwlock_id, thread_id)?;

// Write lock
rwlock::syscalls::sys_rwlock_wlock(&object_manager, rwlock_id, thread_id)?;

// Unlock
rwlock::syscalls::sys_rwlock_unlock(&object_manager, rwlock_id, thread_id)?;
```

## File system operations

File system operations are handled through the VFS:

```rust
use oc_lv2::fs;

// Open file
let fd = fs::syscalls::sys_fs_open(&vfs, "/dev_hdd0/game/data.bin", flags, mode)?;

// Read
let bytes_read = fs::syscalls::sys_fs_read(&vfs, fd, buffer, size)?;

// Write
let bytes_written = fs::syscalls::sys_fs_write(&vfs, fd, buffer, size)?;

// Seek
fs::syscalls::sys_fs_lseek(&vfs, fd, offset, whence)?;

// Close
fs::syscalls::sys_fs_close(&vfs, fd)?;

// Stat
let stat = fs::syscalls::sys_fs_stat(&vfs, "/dev_hdd0/game/data.bin")?;
println!("File size: {} bytes", stat.size);
```

## Time management

```rust
use oc_lv2::time;

// Sleep
time::syscalls::sys_timer_sleep(seconds)?;
time::syscalls::sys_timer_usleep(microseconds)?;
```

## Example: Custom syscall handler

```rust
use oc_lv2::{SyscallHandler, syscall_numbers::*};
use oc_memory::MemoryManager;
use oc_vfs::VirtualFileSystem;
use std::sync::Arc;

// Create subsystems
let memory = MemoryManager::new()?;
let vfs = Arc::new(VirtualFileSystem::new());

// Create handler
let handler = SyscallHandler::with_vfs(vfs);
handler.set_emulator_memory(memory);

// Handle a syscall
let args = [0; 8];
match handler.handle(SYS_PROCESS_GETPID, &args) {
    Ok(pid) => println!("Process ID: {}", pid),
    Err(e) => eprintln!("Syscall failed: {:?}", e),
}
```
